# Adjudication Template

- Task ID:
- Domain / difficulty:
- Annotator A decision:
- Annotator B decision:
- Check disagreements:
- Disagreement category:
- Final decision:
- Required edits:
- Notes for paper subset:

Allowed final decisions: `accept`, `revise`, `reject`.

Suggested disagreement categories:

- `criterion_scope`: missing, redundant, or over-broad gold criterion.
- `donespec_mapping`: DoneSpec does not encode the intended criterion.
- `near_miss_validity`: near miss is invalid, too easy, or cites the wrong criterion.
- `trace_plausibility`: reference trace skips required observation or violates policy.
- `templating_or_duplicate`: task is too templated or duplicative.
- `severity_threshold`: annotators agree on issue but disagree on accept/revise/reject severity.
